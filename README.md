# 🛡️ Credit Card Fraud Detection Using Machine Learning

Machine Learning project to detect fraudulent credit card transactions using multiple classification models with class imbalance handling and performance comparison.

Developed in Google Colab using Python.

---

## 📌 Overview

This project builds a fraud detection pipeline using supervised machine learning techniques.  
Since fraud datasets are highly imbalanced, SMOTE is used to improve model learning.

Models compared:
- Logistic Regression
- Random Forest
- XGBoost
- LightGBM
